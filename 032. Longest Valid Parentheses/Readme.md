遇到 '('  push 进栈，遇到 ')' 时，进行判断，如果栈为空，更新 leftmost；如果不为空，pop 元素，pop 后为空，maxlen = max(maxlen, i-leftmost)，pop 后不为空，maxlen = max(maxlen, i-stack[-1])；

当遇到 ')' 且栈为空时，更新 leftmost 为当前的索引值。

`s = (()))(()`

|      | idx  | leftmost | stack | max  |
| ---- | ---- | -------- | ----- | ---- |
| (    | 0    | -1       | 0     | 0    |
| (    | 1    | -1       | 0,1   | 0    |
| )    | 2    | -1       | 0     | 2    |
| )    | 3    | -1       | E     | 4    |
| )    | 4    | 4        | E     | 4    |
| (    | 5    | 4        | 5     | 4    |
| (    | 6    | 4        | 5, 6  | 4    |
| )    | 7    | 4        | 5     | 4    |
